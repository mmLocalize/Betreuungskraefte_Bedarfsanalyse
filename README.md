# Betreuungskraefte_Bedarfsanalyse
Übungsprojekt Datenanalyse zum aktuellen Bedarf und Angebot von Betreuungskräften in der Seniorenbetreuung

# Ausgangsfragen: 
- Wie hoch ist der Bedarf an Betreuungskräften aktuell und in Zukunft?
- Wie viele ausgebildete Betreuungskräfte gibt es?
- Welche Weiterbildungsangebote, welche damit verbundenen Kosten, wie hoch ist das Interesse daran?
- Würden mehr potenzielle Interessenten die Weiterbildung machen, wenn Träger der Betreuungseinrichtungen die Kosten übernehmen? Welches Potenzial?

# Mögliche Datenquellen:
destatis
Kranken- und Pflegekassen
Stellenanzeigen
Weiterbildungsanbieter
Große Träger wie Diakonie, Malteser usw.
